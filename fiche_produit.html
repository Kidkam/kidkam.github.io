Counter-Strike : Global Offensive

  • Editeur(s) / Développeur(s) : Hidden Path | Valve
  • Sortie France : 21 Août 2012
  • Genre(s) : FPS
  • Thème(s) : Contemporain
  • Classification : +16 ans
  • Mode(s) : Multi en ligne
  • Modèle(s) économique(s) : Payant à l'acquisition
Description : Counter-Strike : Global Offensive est un FPS multijoueurs en ligne sur PC. Les anti-terroristes et les terroristes s'affrontent dans différents modes de jeu avec une trentaine d'armes différentes, sans compter les grenades. Un mode entraînement permet aux joueurs de s'habituer aux commandes.

Site à but non lucratif réalisé dans le cadre d'un projet d'étude en IUT ASPE Lyon 1
